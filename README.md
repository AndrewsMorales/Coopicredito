# Coopicredito

Este proyecto fue desarrollado en **Drupal** utilizando **JavaScript**, **jQuery**, **PHP** y **Twig** para proporcionar dos funcionalidades principales: una **intranet corporativa** con un **calendario avanzado** y un sistema de **gestión de reservas** de salas y activos, además de un **sistema de solicitud de crédito** para la gestión de datos relacionados con préstamos.

## Características Principales

### 1. **Intranet con Calendario Avanzado**
- **Eventos Globales y Personales:** Permite crear eventos tanto a nivel organizacional (globales) como a nivel de usuarios específicos (personales), facilitando la coordinación y gestión de actividades.
- **Reservas de Salas y Activos:** Los usuarios pueden reservar salas como la **sala de juntas**, con configuraciones personalizables como **sillas (40), pantallas, papelería** y **servicios de comida**, optimizando el uso de los recursos compartidos.
- **Interfaz Intuitiva:** El calendario y las reservas están diseñados para ofrecer una experiencia fácil de usar, permitiendo una administración eficiente de los eventos y los recursos.

### 2. **Sistema de Solicitud de Crédito**
- **Ingreso de Datos Detallados:** Los usuarios pueden completar formularios con información necesaria para solicitar un crédito, incluyendo:
  - Tipo de crédito solicitado.
  - Datos personales del solicitante.
  - Información sobre **garantías**, **ahorros**, **deudas**, **codeudores** y **prendas de vehículos**, entre otros.
- **Análisis Completo:** El sistema procesa y gestiona la información para evaluar la viabilidad del crédito solicitado, garantizando una evaluación precisa y eficiente de cada solicitud.
- **Integración y Validación de Datos:** Se implementan procesos automáticos para la validación de la información ingresada, asegurando la precisión en el proceso de solicitud.

## Tecnologías Utilizadas
- **Backend:** Desarrollado en **Drupal**.
- **Frontend:** Usando **JavaScript**, **jQuery** para interactividad, y **Twig** para las plantillas.
- **Base de Datos:** MySQL para la gestión de usuarios y contenido.
- **PHP:** Usado para la lógica del servidor y la integración con Drupal.

## Funcionalidades del Proyecto:
- **Intranet en Drupal** con gestión de eventos y recursos internos.
- **Calendario Avanzado** para la creación de eventos globales y personales.
- **Gestión de Reservas** de salas y activos con configuraciones personalizadas.
- **Sistema de Solicitud de Crédito** para la recopilación y análisis de información financiera.
- **Interfaz Amigable** que permite a los usuarios navegar fácilmente y realizar tareas de manera eficiente.

## Conclusión

Este proyecto destaca por su capacidad de integrar múltiples funcionalidades dentro de una sola plataforma. La combinación de herramientas como **Drupal**, **JavaScript**, **jQuery**, **PHP** y **Twig** asegura una solución robusta para la gestión interna de recursos y un sistema de solicitud de crédito eficiente para la toma de decisiones financieras.
